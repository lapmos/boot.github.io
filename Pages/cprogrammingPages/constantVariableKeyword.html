Constants, Variables, Keywords in C Programming

Constant :

Let's see what are the constants. Constant is the thing that remains the same throughout the enviorement is not possible to change.

Variables :

Variables which are used when we have a store any value. So that we store it in any one particular name that what can be changed and if we have to change it anywhere, we can change it so that what we call a variable.

Example : a = 6, a = 17

Here 'a' is variable and 6 and 17 are the constant.

Which 'a' variable is what will be doing in our system will be carrying a same location.

The first the value of 'a' it was 6 and later the value of 'a' is 17. its value is overlocated. It is located at the same location right now But 'a' value is overwritten, and Now value of 'a' is '17'. And what happened in that, is that which we have variables names, the variable name 'a' is same but ths variable value changed. So 'a' is called variables and 6 and 17 are called constant.

Rules of Making Variables Name

  1. The first character must be an alphabet and underscore (_).
  2. No commas, No blanks allow.
  3. Special symbol is not allowed.
  4. Variable Name are Case sensitive.

Types of Constant

Integer Constant :

Integer constant they ca be one digit, numbers, Octal Number etc.

Rules of Making Interger Constant

  1. An Interger Constant must have at least one digit.
  2. It must not have a decimal point.
  3. It Can be positive and negative.
  4. Default sign is positive.

    5. Ex: 245, 786, 321, -564, -3001 etc.

Rules of Making Real Constant

  1. A Real Constant must have at least one digit.
  2. It must have a decimal point.
  3. It Can be positive and negative.
  4. Default sign is positive.

    5. Ex: 3.42123, 3.14, 0.89374, -564.8893, -3001.000 etc.

Character Constant

When any character contains enclosed within '8' (a pair of single quote) and any alphabet.

Note : '8' is not equal to 8. Because '8' it is a character Now. And 8 it is a Number/digit/integer.

Ex : '5', 'p' , 'q' etc.

String Constant

When more than one character is enclosed inside double quote then called string constant. And include letters, special character and digits

NOW UNDERSTAND

Note : 'w' is a character and "w" is string. So both are different.

Ex : "lapmos", "Hello World" , "2021" etc.

Keywords

Keywords are already reserved Words in c programming language. The compiler knowns these words already. The keywords cannot be used as variable names because if we do so we are trying to assign a new meaning to the keyword, which is not allowed by the computer.There are 32 keywords in C programming language.
